# POST /account/link
Привязывает аккаунт к соц. сети, используя
указанный код в запросе, который был выслан
игроку в ответ на команду /link.

Требуется валидный токен в заголовках запроса.

Тело запроса в JSON:
```
{
    "code": "A83kAMz"
}
```

Пример ответа:
```
{
    "type": "VK",
    "user": 460100799
}
```
